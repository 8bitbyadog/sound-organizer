# Sound Organizer

A tool for organizing and analyzing sound files for music production, podcasting, and audio work.

## Features

- Browse and select audio files
- Play audio files with a built-in player
- Tag and categorize sounds
- Search and filter by name or tag
- Simple and intuitive interface

## Project Structure

The project consists of two main components:

1. **Simple Web-Based Version**
   - `simple-audio-organizer.html`: A standalone HTML file with JavaScript that can be opened in any browser
   - Features basic audio organization and playback capabilities
   - No installation required

2. **Full Desktop Application** (in development)
   - Electron-based desktop application
   - Python backend for advanced audio analysis
   - More advanced features like BPM detection, key detection, and instrument classification

## How to Use the Simple Version

1. Open `simple-audio-organizer.html` in your web browser
2. Click the file input button to select audio files from your computer
3. Use the player controls to play, pause, and stop audio
4. Select a file and click "Add Tag" to categorize it
5. Use the search box to filter files by name or tag
